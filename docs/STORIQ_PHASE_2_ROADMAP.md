# StorIQ Phase 2 Roadmap

Document only. Do not build in MVP.

## Persistence And Access

- Supabase login and database persistence.
- Role-based access for internal users.
- Version history per location.
- Facility database sync from Supabase or another approved system of record.

## AI And Automation

- AI generation API integration.
- Bulk generation queue.
- Section-level AI regeneration controls with saved copy versions.
- Before/after SEO audit comparison.
- Client approval workflow.

## Location Data Verification

- Google Maps Places/Distance API for real 10-mile validation.
- Automated scheduled import of all facilities from the company master source.
- More complete storage/image library management.

## Published Page QA

- Live URL crawler for published Storagely pages.
- Lighthouse/PageSpeed integration.
- Rich results/schema testing helper.

## Publishing

- Direct Storagely publishing only if API/access exists.
