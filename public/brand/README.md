# Brand images

Paste your StorIQ logo and favicon files here.

## Expected filenames

| File | Used for |
|------|----------|
| `brand-logo-header.png` | App header (recommended: wide logo, ~118×40px or similar aspect) |
| `brand-logo.png` | Full logo asset |
| `favicon-32.png` | Browser tab icon (32×32) |
| `favicon.svg` | Optional vector favicon |

The app loads these paths automatically after you paste the files here (no need to copy to `public/` root).

Source EPS (if any) can live here as `brand-logo-source.eps`.
